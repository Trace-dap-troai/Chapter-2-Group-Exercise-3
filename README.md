# Neural Network Classification Project

## Objective
This project aims to design, implement, and evaluate a Feedforward Neural Network (MLP) for a binary classification task.

## Dataset
Breast Cancer Wisconsin Dataset

- 30 numerical features  
- Binary classification (Benign vs Malignant)

## Model Architecture
- Input layer: 30 features  
- Hidden Layer 1: 64 neurons (ReLU activation)  
- Hidden Layer 2: 32 neurons (ReLU activation)  
- Output Layer: 1 neuron (Sigmoid activation)

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  

## Results
The model achieved high performance with strong classification capability on the test dataset.
